## Версионность проекта

Версия проекта соблюдает стандарт Semantic Versioning (SemVer). Каждая версия имеет следующий формат:

- MAJOR.MINOR.PATCH, где:
  - MAJOR: Изменяется при внесении значительных изменений API или несовместимости с предыдущими версиями.
  - MINOR: Меняется при добавлении новых функциональных возможностей, поддерживающих обратную совместимость.
  - PATCH: Обновляется при выпуске патчей, исправляющих мелкие проблемы и баги, не влияющих на основной функционал.
